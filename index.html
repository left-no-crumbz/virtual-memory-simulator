Virtual Memory Simulator

Visualize how FIFO (First In First Out) and LRU (Least Recently Used) page replacement algorithms work in virtual memory management

Least Recently Used (LRU): The page that hasn't been used for the longest time is replaced. This algorithm performs better than FIFO but requires tracking when each page was last accessed.
First In First Out (FIFO): The oldest page in memory is replaced when a new page needs to be loaded. This algorithm is simple but may replace frequently used pages.
